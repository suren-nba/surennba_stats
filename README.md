# NBA 数据工程工具库 (NBA Data Spiders)

这是我的 NBA 数据抓取开源项目！本仓库主要用于托管 [surennba.com](https://www.surennba.com) 网站的核心数据源抓取脚本。

本项目完全基于 **Python** 开发，目的是帮助大家获取NBA的数据，那么我个人网站的所有数据都在这里了。所有抓取逻辑均直接对接 NBA 官网原生 API 以及 pbpstats网站的抓包。

---

## 项目优势

* **国内体感时间优化**：针对中美 12-15 小时时差，脚本内部已将美国东部时间（ET）的比赛日期自动 `+1d` 转换为国内球迷的体感更新时间，确保你的看板数据不过时。
* **现代数据存储**：核心高阶数据采用 **Apache Parquet** 格式进行本地持久化。比传统的 CSV 读取速度快数十倍，且无缝兼容 R 语言（`dplyr` / `arrow` / `duckdb`）进行秒级大数据分析与 Shiny 部署。
* **高鲁棒性架构**：针对常规赛、附加赛（Play-In）、季后赛设计了自动化循环抓取逻辑，并内置异常捕获（Try-Catch），自动跳过尚未开始的比赛阶段，拒绝程序崩溃。
